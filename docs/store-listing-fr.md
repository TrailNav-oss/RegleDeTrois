## Description complète
Proportio est votre outil de calcul du quotidien : proportions, pourcentages, conversions et mise à l'échelle de recettes.

★ RÈGLE DE TROIS
Remplissez 3 valeurs, la 4e se calcule automatiquement. Idéal pour les proportions, les dosages et les calculs du quotidien. Historique consultable avec recherche.

★ RECETTES
Créez vos recettes avec leurs ingrédients. Changez le nombre de portions et toutes les quantités se recalculent. Boutons multiplicateurs rapides (×0.5, ×2, ×3). Mode ajusté : modifiez un ingrédient et les autres s'adaptent. Partagez vos recettes en un clic.

★ POURCENTAGES
5 modes de calcul :
• % d'un nombre (combien font 15% de 200 ?)
• Variation (de 80 à 100, quel % ?)
• Augmentation (200 + 20% = ?)
• Diminution (200 - 20% = ?)
• Trouver le % (30 est quel % de 200 ?)
Raccourcis pour les pourcentages courants (5%, 10%, 15%, 20%, 25%, 50%).

★ CONVERSIONS
Poids (g, kg, oz, lb), volumes (ml, cl, L, cup, fl oz, c.à.s, c.à.c) et températures (°C / °F). Conversion en temps réel. Inversez les unités en un tap.
